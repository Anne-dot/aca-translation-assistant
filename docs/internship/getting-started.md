## Kuidas alustada

### 1. Repo kloonida
```bash
git clone git@github.com:Anne-dot/aca-translation-assistant.git
cd aca-translation-assistant
```

### 2. Loe läbi
- `README.md` - projekti ülevaade
- `TODO.md` - praegused ülesanded
- `PROJECT_OVERVIEW_DRAFT.md` - täielik roadmap
- `docs/ai-instructions/instructions.md` - koodimise põhimõtted

### 3. Terminite review skript
```bash
python3 src/interactive_glossary_terms_review.py
```

### 4. Andmed
- `data/1_extracted/foundation_glossary.json` - 334 terminit
- `data/ACA_WSO/` - algallikad
